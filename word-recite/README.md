# T0mTalking 单词学习站

直接双击 `T0mTalk.html` 即可使用，不需要安装软件或连接网络。

## 已内置

- 动物学词书：18 章，265 个词汇
- 背诵卡、拼写练习、浏览器英文朗读
- 拼写后停留显示对错、你的答案与正确答案
- 章节筛选、打乱、掌握/待巩固标记、学习进度
- 按日期查看拼错单词和待巩固词汇
- 多词书创建、切换和删除

## 导入其他词书

进入“词书”→“新增词书”，可以直接粘贴词汇，或导入 TXT、CSV、JSON。

文本与 CSV 推荐每行使用以下格式：

```text
abandon | 放弃；抛弃 | abandon a plan
ability | 能力 | the ability to do something
```

也支持制表符或英文逗号分隔。JSON 支持对象数组，字段可使用 `term/meaning/note`、`word/translation/example` 或 `english/chinese/hint`。

词书和学习进度保存在当前浏览器中。清理浏览器站点数据会同时删除自定义词书。
